# fiti_bot
телеграм-бот для фитотерапии

бот сделан для ответа на часто задаваемые вопросы по фитотерапии. а также для рекомендаций по выбору сборов для лечения от тех или иных болезней. 
Используем Telegram API, Python,  fuzzywuzzy (для нечеткого сравнения) и pymorphy2 (для лемматизации).
Бот работает следующим образом: запрос пользователя лемматизируем и нечетко сравниваем его с вопросами из базы знаний. выбирается наиболее близкий вопрос и пользователю  отправляется ответ. На данный момент вместо базы с ответами выложена "заглушка" faq.json вместо которой можно использовать любую свою базу знаний.
